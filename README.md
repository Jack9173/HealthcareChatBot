# HealthcareChatBot

# 🚑 Healthcare Assistant Chatbot 💬  

A **healthcare chatbot** powered by **Mistral-7B-Instruct v0.3** and the **Hugging Face API**, built with **Streamlit**. This chatbot assists users with healthcare-related queries, provides symptom guidance, and converts responses to speech.

## 🌟 Features  
✅ **AI-powered healthcare assistant** using Mistral-7B v0.3  
✅ **Text-to-speech (TTS)** for voice responses  
✅ **Streamlit-based UI** for easy interaction  
✅ **Hugging Face API** implementation (No local model hosting needed)  

## 🚀 Installation  

### 1️⃣ **Clone the Repository**
```sh
git clone https://github.com/your-username/Healthcare-Chatbot.git
cd Healthcare-Chatbot
🛠 Tech Stack
Language Model: Mistral-7B-Instruct v0.3
Framework: Streamlit
Text-to-Speech (TTS): gTTS
API Integration: Hugging Face Inference API
Programming Language: Python
📌 How It Works
The chatbot processes user queries related to healthcare.
The Mistral-7B model provides an intelligent response.
The response is displayed in text and converted to speech using gTTS.
Users can listen to the response directly in the app.
🖼 Example Usage
💬 User: "I have a fever and sore throat. What should I do?"
🤖 Chatbot: "You might have a cold or flu. Stay hydrated and get plenty of rest. If symptoms persist, consult a doctor."
🔊 (Audio response available!)
